Hello!

Welcome to my demonstration of Network Address Translation and port forwarding!

If you're seeing this page then the process is working. Here's what we did to get you here:

  1. I set up a DHCP reservation on my local (home) router for my Raspberry Pi (which you are connected to).
  2. I set up port forwarding on the router to forward inbound TCP packets for port 8080 to the Raspberry Pi's internal IP address (192.168.0.252) on port 5000.
  3. I set up a simple Flask webapp on the Raspberry Pi to listen out for your HTTP request on port 5000 and respond with this page.
